## Project Description

This project is a real-time chat application built using Node.js, Express, Socket.io, and MongoDB.  
It allows users to sign up, log in, join chat rooms, send messages, and communicate in real time.

All messages are stored in MongoDB for persistence, and users can see typing indicators while chatting.

---

## Features Implemented

The following features have been implemented as per lab test requirements:

- User Signup with MongoDB storage  
- User Login with session stored using localStorage  
- Predefined chat rooms  
- Real-time messaging using Socket.io  
- Room-based chat functionality  
- Typing indicator feature  
- Message storage in MongoDB  
- Logout functionality  
- GitHub repository with proper project structure
